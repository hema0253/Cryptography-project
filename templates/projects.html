About the KDD Dataset

Our network intrusion detection system is trained on the KDD Cup 1999 dataset, which contains a wide variety of simulated intrusions in a military network environment.

The dataset includes various types of attacks:

  • Denial of Service (DoS)
  • Probe attacks
  • User to Root (U2R)
  • Remote to Local (R2L)

Our model achieves high accuracy in detecting these attack types and can be used for real-time network monitoring.
